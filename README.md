# File Parser

A simple file parser application written using SpringBoot, Gradle, jUnit, Apache Commons (Lang, Collections e CSV).

## Installation

- `git clone https://github.com/Danilo-Araujo-Silva/file-parser.git`
- `cd file-parser`
- `./gradlew test`
